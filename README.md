# びわ湖電池 Project

**目的**  
びわ湖を下池にした大容量揚水発電を目指します。

**主な構想要素**  
- 上池ダム
- 配管
- 発電所
- びわ湖（下池）

**今後の予定**  
- 上池立地コンテスト
- 環境アセスメント
